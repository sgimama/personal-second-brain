# Personal Second Brain

Vault personal de [Obsidian](https://obsidian.md) organizado con el método **PARA** de Tiago Forte.

## Estructura

| Carpeta | Propósito |
|---|---|
| `00 Inbox` | Captura rápida. Todo entra aquí antes de clasificarse. |
| `01 Projects` | Esfuerzos con un objetivo y una fecha límite. |
| `02 Areas` | Responsabilidades continuas sin fecha final (salud, finanzas, trabajo…). |
| `03 Resources` | Temas de interés y material de referencia. |
| `04 Archive` | Elementos inactivos de las otras tres categorías. |
| `Daily` | Notas diarias. |
| `Templates` | Plantillas reutilizables. |

## Flujo

1. Captura cualquier idea en `00 Inbox`.
2. Revisa el inbox periódicamente y mueve cada nota a `Projects`, `Areas` o `Resources`.
3. Cuando algo deja de estar activo, va a `04 Archive`.

## Sincronización

Este vault se versiona con **git** (`sgimama/personal-second-brain`) y vive fuera de iCloud
para evitar conflictos de sincronización de Obsidian.
